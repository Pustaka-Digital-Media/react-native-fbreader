# react-native-fbreader

RN plugin for FBReader SDK

## Installation

```sh
npm install https://github.com/bakhtiyork/react-native-fbreader.git
```

## Usage

```js
import FBReaderView from "react-native-fbreader";

// ...

<FBReaderView  
  book={filePathToBook}
  textColor="#333"
  fontSize="20"
  background="wood"/>
```
