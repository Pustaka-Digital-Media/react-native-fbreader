# react-native-fbreader

RN plugin for FBReader SDK

## Installation

```sh
npm install react-native-fbreader
```

## Usage

```js
import FBReaderView from "react-native-fbreader";

// ...

<FBReaderView  
  book={filePathToBook}
  textColor="#333"
  fontSize="20"
  background="wood"/>
```
